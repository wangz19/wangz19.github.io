    • Evaluating the hypothesis

      The hypothesis may be overfitting the training sets (~70% data set).

      Evaluation of test error:

      • Linear regression: $$J_{test}(\theta) = 1/{2m_{test}}\sum_{i=1}^{m_{test}}(h_\theta(x_{test}^{(i)}-y_{test}^{(i)})^2])$$

      • Classicfication: $$J_{error}(\theta) = 1/{m_{test}}\sum_{i=1}^{m_{test}}err(h_\theta(x_{test}^{(i)}),y_{test}^{(i)})$$

        where, $$err(h_\theta(x),y) = 1$$, if $h_{\theta}(x)> 0.5$ $and$ $y= 0$ or $h_{\theta}(x)<0.5$ $and$ $ y= 1$ ( make sure how to write otherwise)

      • Diagnosing bias or variance
